Equity Valuation Kit
================================================

Evkit is a library for financial statement analysis, equity valuation and research.
It allows user to download, and parse financial statement of _all_ public companies
listed on stock exchanges in the U.S.

Evkit uses curated data sources and cohesive methodologies,
which allows user to perform equity research, on par with professional quantitative researchers.
